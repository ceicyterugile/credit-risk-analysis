# Mortgage Credit Risk & Default Distribution Analysis

##  Project Overview
This data analytics project focuses on exploratory data analysis (EDA) and cleaning of transactional mortgage tracking datasets. The core objective is to uncover behavioral patterns and statistical distributions that correlate with credit risk and user loan defaults.

##  Tech Stack & Libraries Used
- **Python**: Core programming logic and error handling.
- **Pandas**: Structural data loading, file parsing, and data manipulation.
- **Matplotlib & Seaborn**: Statistical data visualization, custom plot generation, and trend graphing.

##  Key Analytical Distributions Evaluated
1. **Loan-to-Value Ratio (LTV)**: Analyzing the distribution and frequency of collateral value relative to total loan amount.
2. **Debt-to-Income (DEBTINC)**: Mapping out distribution thresholds to identify heavily leveraged borrower clusters.
3. **Loan Default Status (BAD)**: Evaluating class distributions between borrowers who successfully paid off balances versus those who defaulted.
